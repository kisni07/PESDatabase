𝐏𝐞𝐫𝐭𝐚𝐧𝐲𝐚𝐚𝐧 𝐲𝐚𝐧𝐠 𝐬𝐞𝐫𝐢𝐧𝐠 𝐝𝐢𝐚𝐣𝐮𝐤𝐚𝐧:
𝟏. 𝐀𝐩𝐚 𝐢𝐭𝐮 𝐏𝐄𝐒𝐃𝐚𝐭𝐚𝐛𝐚𝐬𝐞?
Sesuai namanya aplikasi ini adalah penyedia layanan database (𝑠𝑒𝑝𝑒𝑟𝑡𝑖 𝑝𝑒𝑠𝑚𝑎𝑠𝑡𝑒𝑟) yang dibuat dan dikembangkan untuk membantu modder dalam membuat patch. Untuk melakukan pengeditan anda membutuhkan tool editor terbaru dari ejogc327.
Saatnya beralih dari Platform Website ke Aplikasi Desktop! Semua kebutuhan anda dalam mencari database eFootball dan PES sudah ada disini.

𝟐. 𝐀𝐩𝐚 𝐛𝐞𝐝𝐚𝐧𝐲𝐚 𝐏𝐥𝐚𝐭𝐟𝐨𝐫𝐦 𝐖𝐞𝐛𝐬𝐢𝐭𝐞 𝐝𝐚𝐧 𝐀𝐩𝐥𝐢𝐤𝐚𝐬𝐢 𝐃𝐞𝐬𝐤𝐭𝐨𝐩?
Banyak Website yang menyediakan layanan database eFootball dan PES seperti pesmaster misalnya, tapi tidak ada yang memberikan fitur lengkap seperti aplikasi PESDatabase yang kami tawarkan.

𝟑. 𝐅𝐢𝐭𝐮𝐫 𝐚𝐩𝐚 𝐬𝐚𝐣𝐚 𝐲𝐚𝐧𝐠 𝐝𝐢𝐭𝐚𝐰𝐚𝐫𝐤𝐚𝐧?
 • 𝗩𝗶𝗲𝘄𝗮𝗯𝗹𝗲 𝗱𝗮𝗻 𝗘𝗱𝗶𝘁𝗮𝗯𝗹𝗲 𝗗𝗮𝘁𝗮𝗯𝗮𝘀𝗲 𝗔𝗰𝗰𝗲𝘀𝘀
Memungkinkan anda untuk mengakses database dalam dua mode, Editable: bisa copy dan paste (akan aktif secara ototmatis jika anda login), Viewable: tidak bisa keduanya (jika anda tidak login).

 • 𝐋𝐢𝐯𝐞 𝐔𝐩𝐝𝐚𝐭𝐞 𝐃𝐚𝐭𝐚𝐛𝐚𝐬𝐞
Setiap ada pembaruan dari KONAMI kami akan langsung memperbarui database dan merilis versi terbaru aplikasi terutama pada bursa transfer musim dingin dan musim panas.

 • 𝐄𝐱𝐩𝐨𝐫𝐭 𝐏𝐥𝐚𝐲𝐞𝐫
Memungkinkan anda untuk mengekspor semua pemain pada database, baik single player atau berdasarkan team.

 • 𝐄𝐱𝐩𝐨𝐫𝐭 𝐓𝐞𝐚𝐦
Memungkinkan anda untuk mengekspor semua Team.

 • 𝐄𝐱𝐩𝐨𝐫𝐭 𝐏𝐥𝐚𝐲𝐞𝐫 𝐀𝐬𝐬𝐢𝐠𝐧𝐦𝐞𝐧𝐭
Memungkinkan anda untuk mengekspor semua Player Assignment, baik berdasarkan Team atau League.

 • 𝐂𝐨𝐧𝐯𝐞𝐫𝐭 𝐞𝐅𝐨𝐨𝐭𝐛𝐚𝐥𝐥 𝐭𝐨 𝐏𝐄𝐒
Seperti yang kita ketahui, base statistik di eFootball relatif lebih rendah jika digunakan untuk game PES, data Motion yang berbeda, ID Pemain dan Nama Jersey yang panjang, Nama Jersey yang menggunakan karakter spesial (ASCII) yang bisa mengakibatkan game crash saat dijalankan dan masih banyak lagi.
Aplikasi ini sudah dibuat cerdas untuk mengkonversi eFootball ke PES, anda cukup gunakan fitur "Export Special" atau "Copy Special" maka system akan otomatis menyesuaikan versi game apa yang akan anda gunakan.

 • 𝐌𝐚𝐠𝐢𝐜 𝐀𝐛𝐢𝐥𝐢𝐭𝐲 𝐆𝐞𝐧𝐞𝐫𝐚𝐭𝐨𝐫
Mengkonversi statistik pemain dari PES lama (PES 2014-2016) ke PES terbaru (PES 2017-2021). Hal ini bertujuan membantu anda untuk tetap bisa menggunakan pemain classic ke dalam patch yang sedang anda buat.
Berikut adalah daftar atribut pada Ability yang dibuatkan secara otomatis agar bisa digunakan pada PES 2017-2021:
PES 2014-2015: Modern GK 5 Stats
PES 2014-2016:  Physical Contact
PES 2014-2019: Thight Position, Agression

 • 𝐂𝐨𝐩𝐲 𝐭𝐨 𝐏𝐄𝐒 𝐄𝐝𝐢𝐭𝐨𝐫
Salin dan Tempel statistik pemain ke PES2020 Editor ejogc327, anda bisa menggunakan 2 mode "Copy" Standard dan "Copy Special".

 • 𝐂𝐥𝐞𝐚𝐧𝐮𝐩 𝐃𝐚𝐭𝐚𝐛𝐚𝐬𝐞 𝐁𝐢𝐧𝐚𝐫𝐲 (𝑪𝒐𝒎𝒊𝒏𝒈 𝑺𝒐𝒐𝒏)

   ✓ Coach.bin
Menghapus database satu-persatu cukup melelahkan bukan? 😁
Dengan file binary yang sudah dibersihkan (tanpa ada data) anda tidak perlu lagi menghapus data satu-persatu secara manual, cukup Impor file CSV hasil Export dati PESDatabase ke Tool Editor. Pengeditan database (CSV) bisa dilakukan sepenuhnya menggunakan Microsoft Excel.

𝟒. 𝐁𝐚𝐠𝐚𝐢𝐦𝐚𝐧𝐚 𝐜𝐚𝐫𝐚 𝐤𝐞𝐫𝐣𝐚 𝐂𝐨𝐧𝐯𝐞𝐫𝐭 𝐞𝐅𝐨𝐨𝐭𝐛𝐚𝐥𝐥 𝐭𝐨 𝐏𝐄𝐒?
Secara khusus kami akan memberikan penjelasan untuk pertanyaan ini, karena ini adalah salah satu fitur unggulan dari aplikasi PESDatabase kami yang tidak akan pernah anda temukan dimanapun kecuali ada orang yang menirunya.

 • 𝐌𝐞𝐧𝐲𝐞𝐬𝐮𝐚𝐢𝐤𝐚𝐧 𝐀𝐛𝐢𝐥𝐢𝐭𝐲 𝐝𝐚𝐧 𝐒𝐭𝐚𝐭𝐢𝐬𝐭𝐢𝐤 𝐏𝐞𝐦𝐚𝐢𝐧
Seperti yang sudah dijelaskan sebelumnya, statistik pemain di eFootball relatif lebih rendah jika dibandingkan dengan seri PES. Salah satu faktor utama mungkin karena perbedaan engine, migrasi dari FOX Engine ke Unreal Engine serta cara KONAMI berjualan. Game diberikan secara gratis tapi kartu pemain yang dijual.
Fitur Export dan Copy "Special" melakukan pencarian data pemain berdasrkan base ID, biasanya satu pemain memiliki beberapa kartu identitas standard, epic dan legend. Dari ketiga variasi kartu tersebut ambil nilai rating tertinggi dan tukar posisi dengan nilai rating terendah.
Hal ini bertujuan untuk menjaga keaslian database dari KONAMI daripada menggunakan formula tertentu yang berdasarkan kira-kira atau berdasarkan selera sendiri. 😁

 • 𝐊𝐨𝐧𝐯𝐞𝐫𝐬𝐢 𝐈𝐃 𝐞𝐅𝐨𝐨𝐭𝐛𝐚𝐥𝐥 (𝟔𝟒-𝐛𝐢𝐭) 𝐤𝐞 𝐏𝐄𝐒 (𝟑𝟐-𝐛𝐢𝐭)
Mengapa ID Pemain di eFootball lebih panjang daripada PES? Seperti yang sudah dijelaskan diatas, hal ini dikarenakan satu pemain memiliki beberapa kartu identitas. Jika kita paksa ID tersebut digunakan pada seri PES maka resiko Game akan crash saat dijalankan.

 • 𝐌𝐞𝐧𝐠𝐚𝐝𝐚𝐩𝐭𝐚𝐬𝐢𝐤𝐚𝐧 𝐅𝐨𝐫𝐦 𝐂𝐨𝐧𝐝𝐢𝐭𝐢𝐨𝐧 𝐞𝐅𝐨𝐨𝐭𝐛𝐚𝐥𝐥 𝐤𝐞 𝐏𝐄𝐒
𝐹𝑜𝑟𝑚 𝐶𝑜𝑛𝑑𝑖𝑡𝑖𝑜𝑛 adalah atribut yang berfungsi untuk menentukan fluktuasi statistik pemain dalam sebuah pertandingan. Seberapa sering seorang pemain akan mendapatkan panah hijau (kondisi bagus) saat pertandingan dimulai. Semakin tinggi angka Form, semakin kecil kemungkinan mereka mendapatkan panah merah (kondisi buruk).
Masalahnya adalah di era PES atribut Form memiliki nilai range 1-8, sedangkan di eFootball berubah drastis dengan nilai range 1-3 (Form: 1=Inconsistent, 2=Standard, 3=Unwavering). Tapi anda tidak perlu khawatir, system kami sudah dibuat cerdas untuk mengatasi hal ini dengan mengkombinasikan 𝑃𝑙𝑎𝑦𝑒𝑟 𝑊𝑒𝑒𝑘𝑙𝑦 dan 𝐹𝑜𝑟𝑚 𝐶𝑜𝑛𝑑𝑖𝑡𝑖𝑜𝑛 pada database.

 • 𝐏𝐞𝐦𝐞𝐭𝐚𝐚𝐧 𝐒𝐤𝐢𝐥𝐥 𝐝𝐚𝐧 𝐏𝐥𝐚𝐲𝐢𝐧𝐠 𝐒𝐭𝐲𝐥𝐞 𝐚𝐧𝐭𝐚𝐫 𝐕𝐞𝐫𝐬𝐢 𝐆𝐚𝐦𝐞
Terkadang hal ini sering dianggap remeh oleh sebagian orang yang tidak terlalu memperhatikan detail kecil pada suatu realitas. Tapi jika kita sudah mengetahui kebenarannya tentu ini akan jadi masalah jika tidak dipetakan dengan benar. Karena hampir pada setiap versi KONAMI selalu mengubah ID dari kedua atribut ini karena penambah skill pemain dan gaya bermain.
System kami sudah dirancang secara khusus untuk memetakan 𝑆𝑘𝑖𝑙𝑙 dan 𝑃𝑙𝑎𝑦𝑖𝑛𝑔 𝑆𝑡𝑦𝑙𝑒 secara otomatis berdasarkan versi game yang akan anda gunakan. Pemetaan ini sudah mendukung untuk 𝗲𝗙𝗼𝗼𝘁𝗯𝗮𝗹𝗹 𝘁𝗼 𝗣𝗘𝗦 dan 𝗣𝗘𝗦 𝘁𝗼 𝗣𝗘𝗦 seri lama (2014-2016) ke seri terbaru (2017-2021).
 
𝟓. 𝐀𝐩𝐚 𝐩𝐞𝐫𝐛𝐞𝐝𝐚𝐚𝐧 𝐄𝐱𝐩𝐨𝐫𝐭 𝐝𝐚𝐧 𝐄𝐱𝐩𝐨𝐫𝐭 𝐒𝐩𝐞𝐜𝐢𝐚𝐥?
Kedua fitur ini memiliki fungsi yang sama, untuk mengekspor database ke file CSV dengan struktur data yang sudah disesuaikan dengan tool ejog327 versi terbaru (V0.12.9).
Perbedaannya terletak pada fungsi dan kebutuhan. Jika anda lebih suka menggunakan statistik pemain yang sama seperti pada eFootball gunakan Export (Standard), tapi jika anda lebih suka statistik pemain yang sudah diadaptasikan untuk seri PES gunakan Export Special. Hal ini berlaku juga untuk fitur Copy to PES Editor.

𝟔. 𝐏𝐄𝐒𝐃𝐚𝐭𝐚𝐛𝐚𝐬𝐞 𝐯𝐬 𝐏𝐄𝐒𝐄𝐝𝐢𝐭𝐨𝐫
Saat ini fokus kami hanya untuk menyediakan layanan database saja, untuk proses editing tetap menggunakan tool "PESEditor ejog327" yang sudah terbukti stabil. Anda tidak perlu khawatir, tool PESDatabase ini sudah disesuaikan dengan PES2020 Editor ejog327, anda cukup impor hasil expor yang sudah dibuat.
Walaupun sebetulnya kami bisa saja membuat tool editor juga, tapi saat ini kami belum terlalu memikirkannya. Mungkin akan jadi bahan pertimbangan pada proyek aplikasi selanjutnya.

Jika ini Gratis kenapa ada Premium?
Pada era eFootball hampir setiap update database KONAMI selalu mengubah letak offset dan bit pada file binary mereka. Tentunya setiap kali ada update database kami harus membedah binary untuk menemukan letak offset dan bit yang baru, dan ini bukanlah pekerjaan yang mudah. 😊
Selain itu, karena ini adalah Aplikasi Dekstop bukan Platform Website kami tidak bisa memasang iklan sebagai sumber pendapatan untuk biaya opersional pengembang.
